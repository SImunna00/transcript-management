# University Logo Setup

## Instructions for adding the NSTU logo:

1. **Download the NSTU logo** from: https://www.nstu.edu.bd/ (official website)
2. **Save the logo** as: `public/images/nstu-logo.png`
3. **Recommended specifications:**
    - Format: PNG (with transparent background)
    - Size: 200x200 pixels or higher
    - Aspect ratio: Square or circular

## Alternative logo sources:

-   University official website
-   University branding guidelines
-   Contact university administration for official logo files

## File path in code:

The PDF template will automatically use the logo if placed at:
`public/images/nstu-logo.png`

If no logo is found, a placeholder will be shown in the PDF.
