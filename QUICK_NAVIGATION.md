# Quick Navigation Guide

## Important Paths

### Image Assets

-   Campus Background: `public/assets/images/campus-background.jpg`
-   Images Folder: `public/assets/images/`

### View Files

-   Home Page: `resources/views/home.blade.php`
-   Layout: `resources/views/layouts/app.blade.php`
-   Authentication Views: `resources/views/auth/`

### Controllers

-   Student Controller: `app/Http/Controllers/StudentController.php`
-   Document Request Controller: `app/Http/Controllers/DocumentRequestController.php`

### Database

-   Migrations: `database/migrations/`
-   Seeders: `database/seeders/`

## VS Code Navigation Tips

1. **Quick Open** (`Ctrl+P`): Type part of a filename to quickly open it
2. **Go to Symbol** (`Ctrl+Shift+O`): Jump to functions/methods within a file
3. **File Explorer** (`Ctrl+Shift+E`): Toggle the file explorer view
4. **Search in Project** (`Ctrl+Shift+F`): Search for text across all files
5. **Terminal** (`Ctrl+` `): Open integrated terminal
